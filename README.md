# 📊 Customer Churn Prediction using ML Pipeline

## 📌 Objective of the Task

The objective of this project is to build a reusable and production-ready machine learning pipeline to predict customer churn using the Telco dataset.

---

## ⚙ Methodology / Approach

- Data preprocessing using Pipeline (scaling + encoding)
- Model training using Logistic Regression and Random Forest
- Hyperparameter tuning using GridSearchCV
- Model evaluation using accuracy and classification metrics
- Exporting pipeline using joblib for reuse

---

## 📈 Key Results / Observations

- The pipeline successfully automated preprocessing and training.
- GridSearchCV improved model performance by selecting optimal parameters.
- The final model achieved strong accuracy in predicting customer churn.
- The exported pipeline can be reused in production environments.
